# WiseMama Token Mapping

- Source JSON: `packages/design-core/design-core.json`
- CSS tokens: `packages/design-core/variables-css.css`
- Runtime CSS import: `src/styles/variables-css.css` loaded in `src/main.jsx`

Current app uses legacy aliases (`--accent-orange`, `--text-main`, etc.), now mapped to Design Core variables in `variables-css.css`.
